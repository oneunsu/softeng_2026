## 과제 목록

### 📂 hw02
파이썬 프로그램 연습

- `check_even.py` : 홀짝 구분하기
- `f2c.py` : 단위 변환 (화씨 ↔ 섭씨)
- `factorial.py` : 팩토리얼 구하기
- `get_primes.py` : 소수 판별하기
- `gugudan.py` : 구구단 출력
- `is_prime.py` : 소수 구하기
- `sum_even.py` : 짝수의 합 구하기

### 📂 hw03
`hw02` 프로그램을 **GUI 기반 프로그램**으로 확장

- GUI 구현: `tkinter` 활용
- 출력 꾸미기: `rich` 활용
- 프로그램 형태로 UI 구성 및 꾸미기

### 📂 hw04
html 기반 **웹 애플리케이션** 코드

- `gugudan_app.py` : 구구단 웹 애플리케이션
- `unit_app.py`: 단위 변환 웹 애플리케이션

### 📂 hw05
개인 홈페이지 제작 프로젝트

- HTML 파일: `index.html`, `about.html`, `vision.html`, `projects.html`
- CSS: `css/style.css`
- JS: `json/script.js`
- 배포용 폴더: docs (저장소 최상단에 위치)
